# Architecture Decision Records

Use this directory for short architecture decision records when a decision has
lasting consequences for contributors.

Recommended filename format:

```text
0001-short-title.md
```

Suggested structure:

```md
# 0001 Short Title

## Status

Accepted

## Context

What problem or constraint forced the decision?

## Decision

What did we decide?

## Consequences

What tradeoffs follow from the decision?
```
